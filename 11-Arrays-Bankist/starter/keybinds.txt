/*

VS Code Operations:
alt+p - show vs code commands
ctrl+p - show recent files and workspaces
ctrl+r - show workbench files
alt+r - reload vs code
ctrl+b - primary side bar
ctrl+g - go to line
ctrl+shift+g - source control
alt+1 (...9) - open editor at index 1 (to 9)

Terminal Operations:
ctrl+/ - create terminal
ctrl+; - code run
ctrl+' - terminal kill

Code:

    Text Manipulation:
    alt+n - insert line below
    alt+/ - comment out
    shift+alt+/ - block comment out
    alt+u - delete left
    ctrl+alt+u - delete left word
    alt+o - delete right
    alt+d - copy lines down
    ctrl+alt+d - duplicate selection
    alt+[ - indent lines
    alt+] - outdent lines
    ctrl+alt+i - move lines up
    ctrl+alt+k - move lines down

    Selection:
    shift+alt+m - expand selection
    shift+alt+, - shrink selection
    shift+alt+. - find next match
    ctrl+shift+alt+. - find all matches

    Movement:
    alt+j - left
    alt+k - down
    alt+l - right
    alt+i - up
    alt+h - go to line's beggining
    alt+; - go to line's end
    ctrl+j - ctrl + left
    ctrl+l - ctrl + right
    ctrl+u - cursor undo
    ctrl+n - cursor redo

    Selection Movement:
    shift+alt+j - shift + left
    shift+alt+k - shift + down
    shift+alt+l - shift + right
    shift+alt+i - shift + up
    ctrl+alt+shift+j - ctrl + shift + left
    ctrl+alt+shift+l - ctrl + shift + right
    shift+alt+h - select and go to line's beggining
    shift+alt+; - select and go to line's end

    Actions/Suggestions:
    alt+s - toggle suggestions widget
    alt+d - toggle suggestion details
    alt+i - select previous code suggestion
    alt+k - select next code suggestion
    alt+n - accept suggestion
    / - hide current inline suggestion
    / - hide code action widget

    Copilot:
    ctrl+alt+shift+c - enable/disable copilot
    ctrl+alt+c - copilot generate suggestions
    ctrl+[ - copilot previous inline suggestion
    ctrl+] - copilot next inline suggestion
    alt+\ - trigger inline suggestion
    ctrl+shift+alt+e - use brush on selected text

    File Movement:
    ctrl+i - scroll line up
    ctrl+k - scroll line down


Extension Operations:
    
    Bookmarks:
    alt+f - toggle bookmark
    shift+alt+f - create labeled bookmark
    alt+w - jump to previous bookmark
    alt+e - jump to next bookmark
    shift+alt+w - expand selection to previous bookmark
    shift+alt+e - expand selection to next bookmark

    Live Server:
    ctrl+shift+alt+s - start live server
    ctrl+shift+alt+q - end live server
    
*/
